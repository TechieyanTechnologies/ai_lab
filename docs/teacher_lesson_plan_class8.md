# Class-8 Data Handling & Visualization - 8-Week Lesson Plan

## Overview

This 8-week curriculum introduces 8th-grade students to data literacy through hands-on experience with real datasets. Students will learn to collect, clean, analyze, and visualize data using the School AI Lab platform.

**Target Audience**: Class-8 students (ages 13-14)  
**Duration**: 8 weeks, 2 sessions per week (theory + lab)  
**Session Length**: 40-60 minutes each  
**Platform**: School AI Lab - Offline Data Visualization Module

## Learning Objectives

By the end of this course, students will be able to:
- Understand what data is and why it matters
- Upload and explore CSV datasets
- Clean and prepare data for analysis
- Create appropriate visualizations for different data types
- Interpret charts and draw meaningful conclusions
- Generate professional reports and presentations
- Apply data literacy skills to real-world problems

## Week-by-Week Breakdown

### Week 1: Introduction to Data & Platform
**Theory Session (40 min)**
- What is data? Real-world examples
- Types of data: numbers, text, categories, dates
- Why data matters in our daily lives
- Introduction to the School AI Lab platform

**Lab Session (60 min)**
- Platform tour and navigation
- Create student accounts and first project
- Upload sample dataset (school marks)
- Explore data preview and basic statistics
- **Deliverable**: First project with uploaded dataset

**Materials Needed**:
- Sample dataset: `school_marks.csv`
- Projector for platform demonstration
- Student computers/tablets

### Week 2: Data Preview & Types
**Theory Session (40 min)**
- Understanding data structure (rows, columns, cells)
- Data types: numeric, categorical, text, date
- Missing values and data quality
- Basic statistical measures (mean, median, mode)

**Lab Session (60 min)**
- Upload weather dataset
- Identify column types and data quality issues
- Calculate basic statistics for numeric columns
- Create simple summary tables
- **Deliverable**: Data summary report

**Materials Needed**:
- Sample dataset: `weather_data.csv`
- Calculator for manual calculations
- Printed data type reference sheet

### Week 3: Data Cleaning
**Theory Session (40 min)**
- Why data needs cleaning
- Common data quality issues
- Strategies for handling missing values
- Data type conversions

**Lab Session (60 min)**
- Use Data Explorer to identify issues
- Fill missing values using different methods
- Convert data types (text to numbers, dates)
- Create cleaned dataset version
- Compare original vs cleaned data
- **Deliverable**: Cleaned dataset with documentation

**Materials Needed**:
- Dataset with intentional quality issues
- Data cleaning checklist
- Comparison worksheet

### Week 4: Visualization Basics
**Theory Session (40 min)**
- Why we visualize data
- Chart types and when to use them
- Elements of good charts (title, labels, colors)
- Common mistakes in data visualization

**Lab Session (60 min)**
- Create first histogram (marks distribution)
- Build bar chart (subject performance)
- Practice with different chart options
- Save and organize visualizations
- **Deliverable**: Portfolio of 3 basic charts

**Materials Needed**:
- Chart type reference guide
- Color palette examples
- Sample charts for inspiration

### Week 5: Relationships & Correlation
**Theory Session (40 min)**
- Understanding relationships between variables
- Scatter plots and correlation
- Positive vs negative correlation
- Correlation vs causation

**Lab Session (60 min)**
- Upload student survey dataset
- Create scatter plot (study hours vs grades)
- Calculate correlation coefficient
- Interpret results and draw conclusions
- **Deliverable**: Correlation analysis report

**Materials Needed**:
- Sample dataset: `student_survey.csv`
- Correlation coefficient calculator
- Real-world correlation examples

### Week 6: Time & Trends
**Theory Session (40 min)**
- Time-series data characteristics
- Working with dates and time
- Trend analysis and patterns
- Seasonal variations

**Lab Session (60 min)**
- Work with weather dataset dates
- Create line chart showing temperature trends
- Identify patterns and anomalies
- Calculate moving averages
- **Deliverable**: Time-series analysis with insights

**Materials Needed**:
- Weather dataset with date column
- Calendar for date reference
- Trend analysis worksheet

### Week 7: Mini-Project
**Theory Session (40 min)**
- Project planning and organization
- Choosing appropriate visualizations
- Storytelling with data
- Report structure and presentation

**Lab Session (90 min)**
- Students choose their own dataset
- Plan analysis approach
- Create 3 different chart types
- Write analysis and conclusions
- Generate final report
- **Deliverable**: Complete data analysis project

**Materials Needed**:
- Student-selected datasets
- Project planning template
- Report template
- Peer review checklist

### Week 8: Presentation & Assessment
**Theory Session (40 min)**
- Effective data presentation techniques
- Handling questions and feedback
- Ethical considerations in data analysis
- Future learning opportunities

**Lab Session (90 min)**
- Student presentations (5-10 minutes each)
- Peer feedback and discussion
- Teacher assessment and grading
- Certificate generation
- Course reflection and evaluation
- **Deliverable**: Presentation and final grade

**Materials Needed**:
- Presentation guidelines
- Assessment rubric
- Certificate templates
- Course evaluation form

## Assessment Strategy

### Formative Assessment (Ongoing)
- Lab participation and engagement
- Quality of data cleaning work
- Appropriateness of visualizations
- Progress in guided labs

### Summative Assessment (Week 8)
- **Mini-Project (60%)**: Complete data analysis with 3+ charts
- **Presentation (25%)**: Clear communication of findings
- **Reflection (15%)**: Understanding of concepts and applications

### Rubric Criteria
- **Data Handling**: Upload, clean, and prepare data appropriately
- **Visualization**: Create clear, accurate, and appropriate charts
- **Analysis**: Draw meaningful conclusions from data
- **Communication**: Present findings clearly and professionally
- **Technical Skills**: Use platform features effectively

- Step-by-step guided instructions
- Peer tutoring opportunities

### For Advanced Students
- Additional datasets and complex analysis
- Advanced visualization techniques
- Real-world data challenges
- Mentoring roles for other students

## Technology Requirements

### Minimum System Requirements
- Python 3.8+ installed
- 4GB RAM minimum
- 1GB free disk space
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Network Requirements
- Offline operation (no internet required)
- Local network for multi-user access (optional)

## Troubleshooting Common Issues

### Technical Issues
- **Slow performance**: Reduce dataset size or close other applications
- **Upload errors**: Check file format and size limits
- **Login problems**: Verify credentials and clear browser cache
- **Chart not displaying**: Refresh page and check data format

### Learning Issues
- **Data confusion**: Use simpler datasets and more examples
- **Chart selection**: Provide decision trees and examples
- **Analysis paralysis**: Give specific questions to answer
- **Technical difficulties**: Pair students or provide more support

## Conclusion

This 8-week curriculum provides a comprehensive introduction to data literacy for 8th-grade students. Through hands-on experience with real datasets and the School AI Lab platform, students develop essential skills for the data-driven world while having fun exploring and visualizing information.

The combination of theory and practical application ensures students not only understand data concepts but can apply them to solve real problems. The offline nature of the platform makes it accessible to all schools regardless of internet connectivity, while the teacher-friendly interface ensures successful implementation by educators with varying technical backgrounds.
